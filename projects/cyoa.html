Reflection

Some challenges I had while working on the CYOA project is having trouble linking the files together. We always used markdown to link for example: [text] (link). For HTML its: text . Since I thought that we were using markdown to link pages, I used markdown and was wondering why it wasn't working. I was having trouble when my patner told me that we had set our project to using HTML to link files.

The takeaway I had was remembering what was markdown and when do we use it, and what is html and when do we use it. By determining the differences of markdown and HTMl, we can know what to use. If were using HTML for projects, the entire code needs to be HTML.

Some next steps are finding out first if were coding a project using markdown or HTML so you wouldn't be confused in the future midway through the project.
